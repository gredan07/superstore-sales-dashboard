# Superstore Sales Performance Dashboard

## Project Overview
Analyzed 9,994 rows of retail sales data to identify sales trends, 
top performing products and regional profitability.

## Tools Used
- Microsoft Excel — Data cleaning & preparation
- Power BI — Dashboard & visualizations

## What I Did
- Removed duplicates and handled null values
- Fixed inconsistent date formats using Text to Columns
- Standardized Sales and Profit columns as numeric values
- Built an interactive dashboard with 4 visuals

## Dashboard Visuals
- Sales by Category (Bar Chart)
- Monthly Sales Trend (Line Chart)
- Top 5 Products by Profit (Table)
- Profit by Region (Donut Chart)

## Key Insights
- Technology category generates highest sales
- West region is most profitable
- Q4 consistently shows peak sales period
